# POWER GYM — одностраничник для спортзалу

Сучасний лендінг на React + Tailwind CSS + Lucide React.

## Важливо

**Не відкривай `index.html` подвійним кліком** — через це виникають помилки CORS і скрипти не завантажуються. Застосунок потрібно запускати через dev-сервер (команди нижче).

## Запуск

```bash
npm install
npm run dev
```

Відкрий у браузері адресу, яку покаже Vite (зазвичай http://localhost:5173).

## Збірка

```bash
npm run build
```

Файли збірки будуть у папці `dist/`.
